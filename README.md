# Frontend Practice Lab

This repository is used to practice and deeply understand
frontend concepts from Junior to Senior level.

## Structure

- css/           → CSS, SCSS, Styled Components
- javascript/    → JS core, async, browser APIs
- react/         → React, hooks, patterns
- performance/   → Web performance & optimization
- tools/         → Build tools, configs

## Rule

- 1 folder = 1 concept
- Every folder must contain a README.md
- Code must explain WHY, not only HOW
